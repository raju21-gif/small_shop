1. Project Overview

ShopManager AI is a modern, web-based inventory and sales management system designed for small businesses (Grocery stores, Medical shops, Retailers). It leverages Artificial Intelligence to predict future stock requirements based on historical sales data, helping owners prevent stock-outs and overstocking.

2. Existing System vs. Proposed System

cancel Existing Manual System

  • Manual counting errors in registers.
  • No forewarning of low stock (Reactive).
  • Time-consuming manual report generation.
  • Risk of physical data loss (damaged books).
  • Owner must be physically present.

check_circle Proposed AI System

  • Real-time automated stock tracking.
  • AI-driven 7-day demand forecasting.
  • Instant sales recording & revenue analytics.
  • Cloud-based secure data (MongoDB Atlas).
  • Remote access via any web-enabled device.

3. System Architecture

The system follows a modern 3-tier cloud architecture:

  • Frontend HTML5, Tailwind CSS, JavaScript
  • Backend API FastAPI (Python), Uvicorn
  • Database MongoDB Atlas (Cloud NoSQL)
  • AI Engine OpenRouter API (LLM Integration)

திட்ட அறிக்கை (Project Report)

1. திட்டத்தின் நோக்கம்

ShopManager AI என்பது சிறு வணிகர்களுக்காக (மளிகைக் கடை, மருந்தகம்) உருவாக்கப்பட்ட ஒரு நவீன மேலாண்மை மென்பொருள் ஆகும். இது தினசரி வியாபாரத்தை எளிதாக்கவும், செயற்கை நுண்ணறிவு (AI) மூலம் எதிர்கால சரக்கு தேவையைத் துல்லியமாக கணிக்கவும் உதவுகிறது.

2. சிறப்பம்சங்கள்

  • AI கணிப்பு: கடந்த கால விற்பனையை ஆராய்ந்து, அடுத்த 7 நாட்களில் எந்தெந்த பொருட்கள் எவ்வளவு தேவைப்படும் என்பதை முன்கூட்டியே தெரிவிக்கும்.
  • உடனடி ஸ்டாக் அப்டேட்: ஒவ்வொரு விற்பனைக்குப் பிறகும் கையிருப்பில் உள்ள சரக்கு தானாகவே குறைக்கப்படும்.
  • பாதுகாப்பான அட்மின் வசதி: வாடிக்கையாளர்கள் ஆன்லைனில் ஆர்டர் செய்யலாம்; அதை அட்மின் சரிபார்த்து உறுதி (Approve) செய்யலாம்.
  • எளிமையான பயன்பாடு: கணினி அறிவு பெரிதாக இல்லாதவர்களும் பயன்படுத்தும் வகையில் எளிமையாக வடிவமைக்கப்பட்டுள்ளது.
  • ஆன்லைன் வசதி: இது இணையத்தில் (Cloud) இயங்குவதால், கடையின் உரிமையாளர் எங்கிருந்தாலும் தனது வியாபாரத்தைக் கண்காணிக்கலாம்.

3. தொழில்நுட்பம்

இந்த மென்பொருள் Python FastAPI மற்றும் MongoDB Atlas போன்ற நவீன தொழில்நுட்பங்களைக் கொண்டு உருவாக்கப்பட்டுள்ளது. இது பாதுகாப்பானது மற்றும் வேகமானது.
